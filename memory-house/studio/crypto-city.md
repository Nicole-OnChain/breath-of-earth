# 🎬 Studio — Crypto City Adventures

_Kids' financial literacy animated series (ages 6-12). Concept/Bible phase._

## Concept
- Disney-style financial literacy cartoon for the digital age
- Core theme: "How do we trust money in a digital world?"
- Main characters: Max (curious), Lily (cautious), Byte (Bitcoin guide)
- Recurring: King Bitcoin, Ethereum the Builder, Speedy Coin, Stable Coin, NFT Artist, The Glitch (villain)
- 10 episodes: money → blockchain → keys → mining → scaling → Lightning → NFTs → altcoins → scams → security
- **Key rule:** NEVER tells kids to buy crypto. Teaches trust, ownership, security, critical thinking

## Revenue
- YouTube → merch (Byte plushie) → courses → activity books
- Parent-safe — every character has strengths AND weaknesses
