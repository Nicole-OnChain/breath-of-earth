# 🌿 Garden — Lessons Learned

_Hard-won knowledge from mistakes and course corrections._

## Portfolio Lessons
1. **Don't let a weak position drift.** TrajectoryRL (DEV=10, flow -13%) sat too long before exit. Act early. (May 8)
2. **Don't buy at breakout peaks.** lium.io +10% in 24h after Stage 2 breakout — wait for pullback, don't chase. (May 3)
3. **Extended positions don't get more money.** Score +53% above 150WMA, Gittensor +28.5% above 30WMA — don't add to extended positions. (May 8)
4. **Social penalty is real.** Gittensor aGap crushed by SOCIAL≥80 — real DEV was elite (105/wk) but score didn't reflect it. (May 3)

## Production Lessons (Breath of Earth)
1. **Seedance "real person" filter** — "young woman" and "man" trigger it. Use fantasy identifiers: "wind-weaver girl," "earth-shaper man." (May 11)
2. **Describe actions architecturally.** "Arm braced on wall beside her head, caging her in" works; "cornered by a man" doesn't. (May 11)
3. **Reference images can be too real.** Photorealistic character refs trigger the filter. Generate painterly refs instead. (May 11)
4. **Style tag goes after outfit description.** "Wearing a plain dark zip jacket over a white tee, dark denim jeans" THEN "cinematic fantasy illustration — painterly" so it controls rendering, not wardrobe. (May 11)
5. **5-6 second clips rush past moments.** 12 seconds per clip gives beats room to land. (May 11)
6. **Music is NEVER in video prompts.** Always added in post via Movie Editor. (May 11)

## Tax Lessons
1. **Negative signs differ by source.** Coinbase CSV, Kraken, and API all handle negatives differently. Normalize per source. (April 2026)
2. **Dust dispositions <0.5 units** → use zero-cost lot. Conservative for IRS. (April 2026)

## Process Lessons

2. **If Nicole asks about something we discussed before,** check the house first. If it's not there, say so and ask her to help fill it in.
